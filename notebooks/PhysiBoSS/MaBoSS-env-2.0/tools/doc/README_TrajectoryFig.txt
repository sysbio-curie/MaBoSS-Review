Display and save a figure showing the probability dynamics of all the states present in a MBSS_FormatTable-generated probtraj_table.csv file.
MBSS_TrajectoryFig.py should be run outside of the folder generated by MBSS_FormatTable.pl and receive one argument: the name of the folder generated by MBSS_FormatTable.pl.

MBSS_TrajectoryFig.py folder_name
